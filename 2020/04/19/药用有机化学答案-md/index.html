第八章 羧酸及其衍生物

¶一、单项选择题

1.既有羧基结构，又有醛基结构的化合物是 （ C ）

Ａ、丙酸 Ｂ、乙酸 Ｃ、甲酸 Ｄ、丁酸

２.乙酸的俗称是 （ B ）

Ａ、蚁酸 Ｂ、醋酸 Ｃ、乳酸 Ｄ、水杨酸

３下列化合物中，酸性最强的是 （ A ）

Ａ、乙二酸 Ｂ、乙醇 Ｃ、乙酸 Ｄ、乙醛

4.下列化合物中，不能使酸性高锰酸钾溶液褪色的是 （ B ）

Ａ、甲酸 Ｂ、乙酸 Ｃ、乙醛 Ｄ、乙醇

5.相对分子质量相近的下列化合物沸点最高的是 （ C ）

Ａ、烃 Ｂ、醇 Ｃ、羧酸 Ｄ、醛

６.丁酸和乙酸乙酯的关系为 （ A ）

Ａ、官能团异构 Ｂ、碳链异构 Ｃ、位置异构 Ｄ、互变异构

７.下列化合物中，属于芳香羧酸的是 （ B ）

Ａ、草酸 Ｂ、肉桂酸 Ｃ、亚油酸 Ｄ、醋酸

８.不能用于鉴别乙酰乙酸乙酯的试剂是 （ D ）

Ａ、溴水 Ｂ、三氯化铁 Ｃ、２,４-二硝基苯肼 Ｄ、希夫试剂

９.下列属于酯水解产物的是 （ A ）

A、羧酸和醇 Ｂ、羧酸和醛 Ｃ、醇和酚 Ｄ、羧酸和酮

10.加热不易发生脱羧反应的是 （ D ）

Ａ、草酸 Ｂ、己二酸 Ｃ、丙二酸 Ｄ、丁二酸

11.关于羧酸下列说法错误的是 （ D ）

Ａ、能与金属钠反应 Ｂ、能与碱反应 Ｃ、能与碳酸钠反应 Ｄ、能与酸反应生成酯

12.苯甲酸及其钠盐可以用作食品和药品的 （ B ）

Ａ、乳化剂 Ｂ、防腐剂 Ｃ、酰化剂 Ｄ、洗涤剂

13.下列化合物中，酸性最弱的是 （ C ）

Ａ、对甲基苯甲酸 Ｂ、对硝基苯甲酸 Ｃ、苯甲酸 Ｄ、间硝基苯甲酸
